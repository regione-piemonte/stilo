# Project Title
**StiloExportMarca** componente di Stilo per l'integrazione con le marche temporali

# Project Description
**StiloExportMarca** componente di Stilo per l'apposizione di una marca temporale con l'integrazione di specifici servizi di provider di firme digitali, 
integrato con il software StiloAggiungiMarca

# Versioning 
Il codice sorgente viene gestito tramite maven e versionato su git.
